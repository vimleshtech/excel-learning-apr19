Excel Terms 
===================
-> Excel file is known is workbook
-> Extension of excel file is 
			.xls		MS Office <2007
			.xlsx		MS Office >=2007
		
-> Excel wokbook is collection of worksheets

-> Worksheet is combination of row and columns 

-> Interaction of row and columns is called cell		

-> Cell address
	Example:
	A1		:here A is col, and 1 is row  no 
	
		
	
Shortcut Keys:
	ctrl+shift + right arrow  : select start to end (all cell in one row)
	ctrl+shift + left arrow  : from right to left 
	ctrl+shift + down arrow  : select one column from top to buttom
	ctrl +shift + top arrow 

			

	ctrl+D 			: copy above cell data to below 
	

	ctrl+shift+ + 		: select and press the key to add new row or column
	ctrl+ -			: to remove selected column or row
